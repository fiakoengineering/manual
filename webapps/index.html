Aplikasi Web (Web Application) adalah aplikasi yang dapat diakses melalui Web Browser. Aplikasi web ini merupakan solusi teknologi untuk memudahkan para praktisi ataupun akademisi dalam menyelesaikan masalah yang dihadapi. Aplikasi web ini juga digunakan sebagai demonstrasi pemanfaatan teknologi bahasa pemrograman python sebagai tool dan skill yang dapat memudahkan pekerjaan.

Per November 2022, terdapat tiga aplikasi web yang telah dikembangkan dan dapat digunakan dalam pekerjaan. Ketiga aplikasi tersebut digabung menjadi satu paket aplikasi web yang dinamai sebagai “Aplikasi Pengolahan Data Hujan”. Artikel mengenai aplikasi bisa baca di halaman Artikel: Aplikasi Pengolahan Data Hujan.
